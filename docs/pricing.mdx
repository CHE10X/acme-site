## Which Tools Do I Need?

### Just getting started?

Start with the free tools. They're genuinely useful, not watered-down:

1. Run **RadCheck** to understand your current reliability posture
2. Use **OCTriage** the next time an incident happens
3. Run **Lazarus** to confirm your recovery assumptions

When free tools keep revealing problems you can't prevent, that's when paid tools pay for themselves.

---

### Running production agents?

**Sentinel + Agent911** is the core production stack:

- Sentinel catches problems before users do
- Agent911 tells you what to do when Sentinel fires

Add **SphinxGate** for routing governance or **Transmission** if you're spending too much on model calls and want automatic cost optimization.

---

### Multi-model or compliance requirements?

**SphinxGate** enforces routing policy and produces an auditable decision log. **Transmission** routes each task to the right model automatically.

---

### Want everything?

**Operator Bundle.** It's the most cost-effective path for teams that know they need full coverage.
