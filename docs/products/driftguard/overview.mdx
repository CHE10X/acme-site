# DriftGuard

Your agent's behavior depends on what it remembers. DriftGuard watches for that memory drifting in ways you didn't intend.

## The Problem

Agent memory isn't static. Context accumulates, tools write state, sessions inject information. Over time, what an agent carries into a conversation diverges from what it should.

This isn't a crash. It's subtler. The agent keeps running. It just starts being slightly wrong — and then more wrong — and you don't notice until behavior has shifted enough to matter.

DriftGuard catches the drift before it compounds.

## What It Watches

**Baseline deviation** — a memory artifact changed in a way that doesn't match expected session-to-session evolution. Could be corruption, an unexpected tool side effect, or something being injected that shouldn't be.

**Context drift patterns** — some change is normal. DriftGuard distinguishes routine accumulation from anomalous shifts.

**Memory integrity posture** — an aggregate signal that feeds into Agent911, so you can see memory health alongside everything else.

## What It Does NOT Do

DriftGuard is a watcher, not a fixer. It doesn't:

- Automatically correct memory artifacts
- Delete or overwrite anything
- Make changes to your agent's context without you deciding to

If it finds something, it flags it. You decide what to do.

## Who Needs This

If you're running short-lived agents on simple tasks, drift risk is low. But if your agents:

- Run for days or weeks with persistent memory
- Share context artifacts across multiple agents
- Need to behave consistently (accuracy matters, not just "did it respond")

...then silent memory drift is a real risk. DriftGuard is the layer that tells you it's happening.

## How It Fits

DriftGuard works alongside Elixir — the memory management layer. Elixir builds and injects the memory surface. DriftGuard watches for that surface diverging over time.

Together: Elixir keeps memory fresh. DriftGuard keeps it honest.

Agent911 surfaces DriftGuard signals in its incident view — so memory integrity shows up alongside liveness and model health, not in a separate place you have to remember to check.
