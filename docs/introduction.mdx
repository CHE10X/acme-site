# Welcome to ACME Agent Supply Docs

Technical documentation for operators running AI agent fleets.

## What We Build

ACME provides the reliability layer your agents need:

- **RadCheck** — Agent health scoring (0-100)
- **Sentinel** — Continuous runtime monitoring  
- **OCTriage** — Entry-point diagnostic terminal
- **Agent911** — Unified incident cockpit (read-only v0.1)
- **Lazarus** — Backup readiness verification
