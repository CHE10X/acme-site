---
title: 5-Minute Quickstart
---

Bring ACME online in observational mode. No configuration changes are performed.

## Steps

1. Clone the repo
2. Run the installer
3. Verify system status in Agent911

```bash
acme_install.sh --apply
```

Expected result:

- RadCheck active
- Sentinel guarding
- Agent911 snapshot available

Observational mode by default. No gateway changes performed.
